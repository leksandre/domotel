  • Примеры анимаций страницы, первого экрана

    • Главная страница с анимацией при прокрутке
      • анимация при прокрутке — v1 (прозрачность);
      • анимация первого экрана — v1 (появление блоков снизу вверх);
    • Главная страница с анимацией при прокрутке
      • анимация при прокрутке — v2 (прозрачность + появление блоков снизу вверх);
      • анимация первого экрана — v2 (прозрачность + блюр + скейл);
    • Главная страница с анимацией при прокрутке
      • анимация при прокрутке — v3 (прозрачность + появление блоков слева);
      • анимация первого экрана — v3 (появление слева + блюр);
    • Главная страница с анимацией при прокрутке
      • анимация при прокрутке — v4 (прозрачность + появление блоков слева-вверх);
      • анимация первого экрана — v4 (появление слева-вверх + блюр);
    • Главная страница с анимацией при прокрутке
      • анимация при прокрутке — v5 (прозрачность + размытие блоков + появление блоков со скейлом);
      • анимация первого экрана — v5 (появление блоков со скейлом + блюр);
    • Главная страница с анимацией при прокрутке
      • анимация при прокрутке — v6 (прозрачность + появление блоков со скейлом по горизонтали);
      • анимация первого экрана — v6 (появление блоков со скейлом по горизонтали + блюр);
