# Размер изображений
Максимальная высота блока с изображением - 480пкс.

Изображение вписывается в размеры блока.

Пропорции изображения используемого в верстке 9:5.

| Брейк             | Размер       |
| (min-width)       | (высота), px |
|:-----------------:|:------------:|
| 320               | 370          |
| 960               | 480          |
